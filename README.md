## Installation
- [x] Download and extract the variantJS repo  

  `git clone https://github.com/dohalloran/variantJS.git`  
- [x] The extracted dir will be called `variantJS-master`  
  `cd variantJS-master`   
- [x] Fire up a local server: `node http_server.js` 
- [x] `localhost:8080`
