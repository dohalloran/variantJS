## Installation
- [x] Download and extract the jVariant repo  

  `git clone https://github.com/dohalloran/jVariant.git`  
- [x] The extracted dir will be called `jVariant-master`  
  `cd jVariant-master`   
- [x] Fire up a local server: `node http_server.js` 
- [x] `localhost:8080`
